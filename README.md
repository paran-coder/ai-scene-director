# AI Scene Director 1.0 RC11

AI Scene Director는 이미지·영상 생성 AI에 전달할 장면을 3D에서 구성하고, AI가 이해하기 쉬운 기준 이미지·Pose·Depth·Mask·프롬프트로 내보내는 프리비주얼라이제이션 도구입니다.

## 사용 흐름

### 1. 장면 만들기

상단의 `1 장면 만들기`에서 자연어로 장면을 입력합니다.

```text
비 오는 밤의 편의점 앞.
검은 코트의 여성과 남학생이 마주 보고 있다.
여성은 우산을 들고 있다.
```

앱은 인물, 소품, 공간, 카메라, 조명과 Shot 초안을 편집 가능한 3D 데이터로 만듭니다.

### 2. 장면 수정하기

상단의 `2 장면 수정하기`에서 핵심 피사체를 선택하고 다음을 수정합니다.

- 인물·소품 위치와 크기
- 포즈와 시선 관계
- 카메라 구도와 렌즈
- 조명 방향과 밝기
- Shot 순서와 Action 시간

`동작 미리보기`는 완성 영상 재생이 아니라 생성 전에 움직임과 카메라 동선을 확인하는 선택형 검수 기능입니다.

### 3. AI용 내보내기

상단의 노란색 `3 AI용 내보내기`에서 생성 목적을 고릅니다.

#### 이미지 생성용

- 기준 이미지
- Pose
- Depth
- 객체 마스크
- 장면·카메라·네거티브 프롬프트
- Shot Manifest

#### 영상 생성용

- 시작·종료 프레임
- 시작·종료 Pose·Depth·Mask
- 동작·카메라 프롬프트
- 샷 길이와 구조 데이터
- Shot Manifest

#### 간단 내보내기

- 최종 프롬프트 복사
- 기준 이미지 다운로드
- 시작·종료 프레임 ZIP
- 전체 자료 ZIP

생성 서비스마다 지원하는 제어 이미지 형식이 다르므로, 기본적으로 기준 이미지와 최종 프롬프트를 사용하고 Pose·Depth·Mask는 해당 서비스가 지원할 때 추가합니다.

자세한 파일 사용법은 `AI_EXPORT_GUIDE.md`를 참고하세요.

## 고급 도구

일반 제작 흐름에 필요하지 않은 기능은 `고급 도구` 안에 있습니다.

- ComfyUI 연결
- 프로젝트 점검과 안전 복구
- JSON 가져오기·내보내기
- 제작 세션 기록
- 저장소 정리
- 집중 모드
- GLB 리그 분석과 수동 본 매핑

ComfyUI는 선택 가능한 로컬 연결이며 서비스 사용에 필수적이지 않습니다.

## 실행

```bash
npm ci
npm run dev
```

프로덕션 빌드:

```bash
npm run build
npm run preview
```

전체 RC 검증:

```bash
npm run verify:rc
```

## 주요 기능

- 자연어 다중 인물·소품·환경·멀티샷 생성
- Scene Base와 Shot Override
- 바라보기·들기·앉기·표면 배치 관계
- 걷기·회전·집기·내려놓기·카메라 Action
- 타임라인 드래그·리사이즈·다중 선택·충돌 검사
- GLB 가져오기·에셋 라이브러리·리타기팅
- 손·다리 IK와 지면 접촉
- 카메라 렌즈·조명·참조 이미지
- 이미지·영상 AI용 자료 출력
- 프로젝트 번들·자동 저장·복구
- 명령 검색·단축키·WebGL 안전 모드

## 현재 제한

- 자연어 분석은 로컬 규칙 기반이라 복잡한 문장은 수동 보완이 필요할 수 있습니다.
- Pose·Depth·Mask 사용법은 연결할 생성 서비스의 지원 범위에 따라 달라집니다.
- 절차형 걷기는 모션 캡처 수준이 아닙니다.
- 얼굴·손가락·립싱크는 1.0 이후 범위입니다.
- 실제 Windows·macOS·Linux 설치 산출물은 외부 CI 검증이 남아 있습니다.
