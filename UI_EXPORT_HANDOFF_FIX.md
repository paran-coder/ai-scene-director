# AI 내보내기 전환 명확화 — RC13

## 문제

사용법 페이지의 `영상 생성용 열기`라는 문구가 외부 영상 생성 서비스가 열리는 것처럼 보였지만, 실제 동작은 영상 AI에 넣을 ZIP 자료 설정 팝업을 여는 것이었습니다.

## 수정

- `이미지 생성용 열기` → `이미지용 자료 만들기`
- `영상 생성용 열기` → `영상용 자료 만들기`
- 카드 버튼은 `이미지용 ZIP 설정 열기`, `영상용 ZIP 설정 열기`로 변경
- 버튼 아래에 `생성 서비스로 이동하는 버튼이 아니라, 업로드할 ZIP 자료 설정을 엽니다.` 표시
- 팝업 제목을 선택 모드에 맞게 동적으로 표시
  - `이미지 생성용 자료 만들기`
  - `영상 생성용 자료 만들기`
  - `간단 자료 내보내기`
- 모드 카드 바로 아래에 생성 사이트로 이동하지 않는다는 설명 표시
- 실행 버튼을 `이미지 AI 자료 ZIP 만들기`, `영상 AI 자료 ZIP 만들기`로 변경
- 사용법에서 설정을 열었다면 닫을 때 사용법 페이지로 복귀

## 검증

브라우저 자동 검사가 다음을 확인합니다.

- 사용법 페이지의 `영상용 자료 만들기` 버튼 클릭
- 영상 생성용 모드가 선택된 팝업 표시
- `영상 생성용 자료 만들기` 제목 표시
- `영상 생성 사이트로 이동하지 않습니다` 안내 표시
- 팝업을 닫으면 사용법 페이지 복귀
