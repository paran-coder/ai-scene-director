## RC15 변경

- 중앙 편집 뷰포트에 `작업 밝기` 토글을 추가했습니다.
- 작업 밝기는 약한 보조광과 노출 보정을 사용하지만 AI용 내보내기 캡처에는 반영되지 않습니다.
- 포인트광은 범위 구체, 스포트라이트는 조사 각도 콘, 방향광은 방향 가이드를 표시합니다.
- 조명 종류별로 실제 적용되는 범위·각도·대상·그림자 항목만 활성화합니다.
- 자유 시점의 마우스 조작 도움말을 접고 다시 열 수 있으며 선택 상태를 로컬에 보존합니다.
- RC14의 이미지·영상 ZIP 무결성 검증과 모드별 Manifest 경로 수정은 그대로 유지합니다.

## 현재 릴리스 게이트

상태: `conditional`

코드 차단 문제는 없습니다. 정식 1.0 승격에는 Windows·macOS·Linux의 실제 브라우저, 설치 산출물, 네이티브 런타임 보고서가 필요합니다.
