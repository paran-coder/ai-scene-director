# AI Scene Director 1.0 RC12 상태

## 제품 흐름

- 1. 장면 만들기
- 2. 장면 수정하기
- 3. AI용 내보내기
- 동작 재생은 `동작 미리보기`라는 선택형 검수 기능으로 표시
- ComfyUI·JSON·진단·세션 기록·리그 설정은 고급 도구로 이동

## 자동 검증

- 자동 테스트: 150/150 통과
- TypeScript·Vite 프로덕션 빌드: 통과
- Tauri 정적 구성 검사: 통과
- Chromium 기본 화면 검사: 통과
- Chromium 1366×768 노트북 검사: 통과
- AI용 내보내기 대화상자 검사: 통과
- 헤더 사용법 버튼 표시·위치·라벨 검사: 통과
- 릴리스 사전검사: 통과

## 현재 릴리스 게이트

상태: `conditional`

코드 차단 문제는 없습니다. 정식 1.0 승격에는 Windows·macOS·Linux의 실제 브라우저, 설치 산출물, 네이티브 런타임 보고서가 필요합니다.
