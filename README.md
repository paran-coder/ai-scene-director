# AI Scene Director MVP 0.10

외부 유료 AI API 없이 로컬에서 실행되는 3D 씬·샷 연출 프로토타입입니다.

## 0.10 핵심 기능

- 로컬 GLB 에셋 라이브러리
  - 인물·소품·환경 GLB 등록
  - glTF 2.0 GLB 헤더 검증
  - IndexedDB에 원본 바이너리 저장
  - 브라우저 재실행 후 자동 복원
  - JSON을 다른 환경에서 연 경우 누락 파일 재연결
- 프록시 객체를 실제 GLB 모델로 교체
  - 기존 Transform 유지
  - Shot Override 유지
  - 객체 관계와 Action 참조 유지
  - Undo·Redo 지원
  - 프록시 모델로 복원
- 가져온 GLB 자동 정규화
  - 원점 중앙 정렬
  - 바닥 높이 정렬
  - 인물·소품·환경 분류별 기준 크기 적용
  - 객체 마스크 렌더링 지원
- 환경 프리셋 수동 교체
  - 편의점 외부
  - 카페 내부
  - 도심 거리
  - 거실
  - 사무실
  - 교실
  - 부엌·식당
  - 기본 스튜디오
- 환경 교체 시 데이터 보존
  - 인물·사용자 소품 유지
  - Shot Override 유지
  - 호환되는 테이블·의자 등의 관계를 새 환경 객체로 재연결
  - 연결 대상을 찾을 수 없는 관계·Action만 안전하게 제거
- 인원수·공간 기준 재배치
  - 인물 1~다수 자동 배치
  - 휴대 소품을 가까운 인물 주변에 배치
  - 차량을 장면 외곽에 배치
  - 카메라를 인물 중심으로 다시 프레이밍
- 0.9 이하 프로젝트를 0.10 스키마로 자동 변환
- GLB 저장·복원·삭제와 환경 교체를 포함한 46개 자동 테스트

## 기존 핵심 기능

- 자연어에서 장소·인물·소품·샷·행동 분석
- 배경 프리셋과 소품 세트 자동 배치
- 인물 역할·의상·외형 메타데이터
- 다중 Entity Scene Graph
- Scene 기본 상태와 Shot별 Override
- Transaction 기반 Undo·Redo
- 기본 휴머노이드 리그·관절 포즈·팔 IK
- lookAt·hold·sitOn·placeOn 관계
- Action Block과 타임라인
- 샷 카메라·시작/종료 프레임·Pose/Depth/Mask 출력
- Shot Package ZIP
- 선택형 로컬 ComfyUI 연결

## GLB 사용 방법

1. 씬 계층에서 인물 또는 소품을 선택합니다.
2. GLB 에셋 라이브러리에서 종류를 선택하고 `GLB 가져오기`를 누릅니다.
3. 등록된 에셋의 `적용`을 누릅니다.
4. GLB 파일이 로컬 저장소에서 사라진 경우 `재연결`로 같은 파일을 다시 지정합니다.
5. `선택 객체를 프록시로 복원`으로 원래 편집 프록시를 다시 표시할 수 있습니다.

## 현재 제한

- 외부 GLB 캐릭터의 Skeleton을 내부 휴머노이드 리그에 리타기팅하지 않습니다. GLB 인물을 적용한 상태에서도 Pose Map은 내부 프록시 리그를 기준으로 생성됩니다.
- GLB 원본 바이너리는 IndexedDB에 저장되므로 JSON 파일만 다른 PC로 이동하면 파일을 다시 연결해야 합니다.
- 텍스처가 GLB 내부에 포함되지 않은 외부 참조형 glTF는 지원하지 않습니다.
- 에셋 제거 시 Undo를 위해 원본 Blob은 즉시 영구 삭제하지 않아 저장소에 고아 파일이 남을 수 있습니다.
- 환경 교체 시 의미가 비슷한 가구를 이름과 태그로 매칭하므로 복잡한 사용자 정의 관계는 수동 확인이 필요합니다.
- 번들 크기가 약 1.33MB이며 코드 분할이 필요합니다.
- 브라우저 기반 프로토타입이며 데스크톱 패키징은 아직 포함하지 않습니다.
