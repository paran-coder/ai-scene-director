# RC15 대표 제작 흐름 사용성 점검

## 점검 기준

- 자연어 장면 적용 직후 편집 대상이 존재하는가
- 첫 수정 대상이 자동 선택되는가
- 사용자가 다음 편집 영역을 찾기 위해 메뉴를 탐색해야 하는가
- 각 연출 단계에서 해당 편집 위치로 직접 이동할 수 있는가
- 화면 공간을 확보하는 집중 모드를 제공하는가

## 시나리오 결과

### 2인 대화

- 첫 수정 대상: 주인공
- 자동 선택: 통과
- 관계: 상호 바라보기 생성
- 샷: 와이드·오버숄더·클로즈업
- 단계 이동: 장면 계층·샷 목록·타임라인·점검·출력 연결

### 제품 연출

- 첫 수정 대상: 핵심 제품·소품
- 불필요한 기본 인물: 생성하지 않음
- 헤드폰·향수·시계·신발·스마트폰 인식
- 제품이 없는 경우 카메라 또는 장면 생성 단계로 안전하게 전환

### 인물 동작

- 첫 수정 대상: 주인공
- 자동 선택과 이동 모드: 통과
- 타임라인 바로가기: 통과
- 걷기·집기·트래킹 Action의 준비 상태 반영

## 클릭 경로 변화

이전:

```text
장면 적용
→ 생성 창 닫기
→ 계층에서 인물 찾기
→ 인물 선택
→ 이동 도구 선택
→ 첫 수정
```

RC15:

```text
장면 적용
→ 핵심 피사체 자동 선택·이동 모드 활성화
→ 첫 수정
```

카메라 또는 타임라인으로 전환할 때는 첫 수정 안내의 바로가기 한 번으로 이동합니다.


## 조명 편집 가독성

- 작업 밝기: 기본 활성화, 사용자 토글과 로컬 상태 보존
- 내보내기 분리: 작업 밝기 보정은 AI용 캡처에 미적용
- 포인트광: 범위 구체 표시, 각도 입력 비활성화
- 스포트라이트: 범위·각도 콘 표시, 대상 선택 지원
- 방향광: 방향 가이드 표시, 범위·각도 입력 비활성화
- 환경광: 세기·색상만 활성화, 범위·각도·그림자 비활성화
- 조작 안내: 접기와 다시 열기 지원
