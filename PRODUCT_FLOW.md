# 제품 중심 제작 흐름

```text
1. 장면 만들기
   자연어 → 편집 가능한 인물·소품·공간·카메라

2. 장면 수정하기
   배치·포즈·시선·카메라·조명·Action 수정
   동작 미리보기는 선택적인 검수 과정

3. AI용 내보내기
   이미지용 또는 영상용 자료 생성
   기준 프레임·제어 이미지·프롬프트·Manifest 제공
```

## 제품 외곽 기능

다음 기능은 핵심 흐름을 지원하지만 일반 사용자 화면의 주 작업은 아닙니다.

- ComfyUI
- 프로젝트 JSON
- 프로젝트 진단과 지원 번들
- 세션 기록
- GLB 리그 수동 보정
- 릴리스·개발 검증 기능

이 기능은 `고급 도구` 또는 관련 고급 패널에서만 노출합니다.
