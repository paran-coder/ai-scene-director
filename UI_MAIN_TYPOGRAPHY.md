# 메인 화면 타이포그래피 개선

RC13의 내보내기 팝업뿐 아니라 메인 편집 화면 전체에 읽기 가능한 글자 크기 기준을 적용했습니다.

## 적용 기준

- 패널 제목: 16px
- 객체·샷 이름: 14px
- 버튼·입력·선택: 13px
- 패널 라벨·타임라인 안내: 12~13px
- 상태·보조 정보: 최소 11px
- 본문 행간: 1.4~1.55

## 적용 영역

- 상단 헤더와 프로젝트 상태
- 씬 계층
- 속성 패널
- 뷰포트 안내와 조작 툴바
- 제작 흐름과 다음 작업 안내
- 샷 카드와 샷 편집기
- 타임라인과 Action 편집기
- 하단 자연어 명령 입력

## 작은 화면

1366×768에서도 핵심 영역을 유지합니다. 낮은 화면에서는 상태 칩과 일부 비핵심 메타 정보만 제한적으로 축소하며, 객체명·버튼·입력·타임라인 본문은 가독성 기준을 유지합니다.

## 자동 검증

Chromium에서 계산된 실제 CSS 글자 크기를 검사합니다. 헤더 버튼, 패널 제목, 객체명, 객체 메타 정보, 샷 메타 정보, 타임라인 안내, 명령 입력이 최소 크기보다 작으면 브라우저 스모크 테스트가 실패합니다.
