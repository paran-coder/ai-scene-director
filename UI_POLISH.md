# UI Polish 적용 요약

## 방향

- 편집 작업을 방해하지 않는 고밀도 다크 대시보드
- 근접 검정 캔버스와 단계가 분명한 카드 표면
- 노란색은 AI 장면 생성, 현재 선택, 핵심 다음 작업에만 사용
- 초록색은 완료·출력 준비, 빨간색은 오류·삭제에만 사용
- 큰 그림자와 과도한 블러 대신 경계선과 표면 명도 차이로 깊이 표현

## 구조 변경

- 헤더를 브랜드·프로젝트 상태·핵심 작업·도구·프로젝트 메뉴로 재구성
- 자주 쓰지 않는 저장·복구·진단 기능을 팝오버 메뉴로 이동
- 계층·뷰포트·속성 패널의 표면과 제목 위계를 통일
- 연출 흐름, 샷 카드, 타임라인의 선택 및 상태 색상 통일
- 온보딩·명령 검색·자연어 생성 모달을 같은 시각 언어로 통일

## 주요 토큰

- Canvas: `#0B0E11`
- Surface: `#181A20`
- Card: `#1E2329`
- Elevated: `#2B3139`
- Primary: `#FCD535`
- Success: `#0ECB81`
- Danger: `#F6465D`
- Text: `#EAECEF`
- Muted: `#929AA5`

## 접근성

- 키보드 포커스에 파란색 2px 링 제공
- `prefers-reduced-motion`에서 전환과 애니메이션 최소화
- 노란 버튼은 검정 텍스트를 사용해 대비 확보
- 색상뿐 아니라 테두리·텍스트·아이콘으로 상태를 함께 표현

## RC13 제품 흐름 재정렬

- 상단의 주요 행동을 `장면 만들기 → 장면 수정하기 → AI용 내보내기`로 제한했습니다.
- 노란색은 최종 핵심 행동인 `AI용 내보내기`에 가장 강하게 사용합니다.
- 동작 재생은 결과물로 오해하지 않도록 `동작 미리보기`로 변경했습니다.
- ComfyUI, JSON, 진단, 세션 기록과 리그 설정은 고급 도구로 이동했습니다.
- 내보내기 창은 이미지용·영상용·간단 모드로 나누고, 사용자가 실제로 받을 파일을 먼저 보여줍니다.
